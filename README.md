## Deploying the project

First, ensure you have node, npm, and firebase-tools installed.

If you haven't, log into firebase.

```console
$ firebase login
```

From the scribble directory, build the app.

```console
$ npm run build
```

Now the app is ready to deploy.

```console
$ firebase deploy
```
